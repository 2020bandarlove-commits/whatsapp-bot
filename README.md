
# لوحة تحكم بوت واتساب (WhatsApp Bot Dashboard)

هذا المشروع عبارة عن واجهة أمامية (Frontend) حديثة واحترافية باللغة العربية لإدارة بوت واتساب. تم تصميمه باستخدام React و Tailwind CSS مع دعم كامل للكتابة من اليمين إلى اليسار (RTL).

## المميزات الرئيسية
- **دعم RTL كامل**: واجهة عربية 100% مع خط "Cairo" الأنيق.
- **إدارة الاتصال**: مراقبة حالة البوت (متصل/منفصل) والربط عبر رمز QR أو كود الربط.
- **إدارة الأوامر (CRUD)**: إضافة، تعديل، وحذف الكلمات المفتاحية وردود البوت التلقائية.
- **سجلات الرسائل والأحداث**: عرض آخر الرسائل الواردة وأحداث النظام في الوقت الفعلي.
- **إرسال رسائل اختبار**: نموذج لإرسال رسائل تجريبية لأي رقم واتساب للتأكد من عمل البوت.
- **نظام الحماية**: صفحة تسجيل دخول مع حماية المسارات باستخدام JWT.

## التشغيل المحلي
1. قم بتثبيت التبعيات:
   ```bash
   npm install
   ```
2. قم بتشغيل خادم التطوير:
   ```bash
   npm run dev
   ```

## المتغيرات البيئية (Environment Variables)
لربط الواجهة مع الباك إند (Backend) الخاص بك، قم بإنشاء ملف `.env` وأضف المتغير التالي:
```env
VITE_API_BASE=https://your-api-url.com/api
```
في حال عدم وجود المتغير، سيقوم التطبيق افتراضياً باستخدام `http://localhost:3000/api`.

## هيكلة الـ API المتوقعة
تعتمد الواجهة على نقاط النهاية (Endpoints) التالية:
- `POST /auth/login`: للمصادقة.
- `GET /status`: لجلب حالة البوت.
- `GET /qr`: لجلب رمز QR كصورة Base64.
- `GET /pairing`: لجلب كود الربط.
- `GET /commands`: لإدارة أوامر الرد التلقائي.
- `POST /send`: لإرسال الرسائل.
- `GET /messages`: لجلب سجل الرسائل.

## ملاحظة المطور
تم تصميم الواجهة لتكون متجاوبة (Responsive) تماماً وتعمل بكفاءة على جميع الأجهزة (موبايل، تابلت، ديسكتوب).
